# tsconfig
<h2>
	<center>global typescript config for my projects</center>
</h2>

## Install
```bash
$ npm install --save-dev @jojopr/tsconfig
```

## Usage
```tsconfig.json```
```json
{
    "extends": "@jojopr/tsconfig",
    "compilerOptions": {
        // ...
    }
}
```
